# AI Study Assistant

## Project Description

This project is an AI-assisted study assistant built using Python.

The system accepts user questions and uses external tools such as:
- calculator tool,
- file reader tool,
- search functionality.

The goal is to demonstrate an AI-based software solution with tool usage.

## Features

- User input handling
- AI agent workflow
- Tool integration
- File reading
- Calculations
- Testing support

## Technologies

- Python
- Git
- GitHub

## Run

```bash
python app/main.py
```
